# ReliefSync - Product Overview

ReliefSync is a full-stack disaster relief management platform that coordinates emergency response between citizens, NGOs, government agencies, and field agents.

## Core Purpose
- Real-time disaster reporting and monitoring with geolocation
- Relief coordination and resource distribution tracking
- Multi-stakeholder communication during emergencies

## User Roles
- **Citizen**: Report disasters, request help, track relief status
- **NGO**: Manage relief tasks, coordinate distribution, QR-based delivery tracking
- **Government Agency (gov_Agent)**: Oversee operations, manage field agents, verify organizations
- **Admin**: Full system access, user verification, resource allocation, analytics

## Key Capabilities
- Interactive maps with heat visualization (Leaflet)
- AI-powered disaster assessment (OpenAI)
- Satellite imagery analysis (Google Earth Engine)
- Real-time updates via WebSockets (Socket.IO)
- Push notifications (OneSignal)
- QR code-based resource tracking
- Web3 wallet integration for transparent resource tracking
- Payment processing (Razorpay)
